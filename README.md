## Reproduce the analysis
First install and start jupyter
```sh
python -m venv .

# for windows 
Scripts\activate
# Linux
source bin/activate

pip install -r requirements.txt
jupyter notebook
```
Then use jupyter (in the browser though) to access to the file.
