## Reproduce the analysis locally 
1. Install the environment
    ```sh
    python -m venv .

    # for windows 
    Scripts\activate
    # Linux
    source bin/activate

    pip install -r requirements.txt
    ```
2. Start jupyter
    ```sh
    jupyter notebook
    ```
3. Use jupyter interface (in the browser though) to access to the file

4. Run the notebook
